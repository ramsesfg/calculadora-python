# 🧮 Calculadora en Python

Calculadora de consola que realiza las 4 operaciones básicas.

## Características
- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División
- ⚠️ Validación de errores (letras, división entre cero)
- 🔄 Funciona en bucle hasta que el usuario salga

## Cómo ejecutar
```bash
python calculadora2.py
```

## Tecnologías
- Python 3
